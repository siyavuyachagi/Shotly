# Changelog

All notable changes to the "shotly" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/)
and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.0.1] - 2026-05-30

### Added
- Initial release
- Smart filename templating — named after the active source file (e.g. `user-service.png`)
- Save with dialog or auto-save (configurable)
- Copy screenshot to clipboard without saving
- Background color customization
- Background presets
- Watermark / branding support
- Show/hide line numbers
- Show/hide window controls
- Show/hide window title
- Rounded corners toggle
- Transparent background support
- Real line numbers support
